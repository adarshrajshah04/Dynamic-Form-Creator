# FormCrafter

FormCrafter is a dynamic form builder built with HTML, CSS, and JavaScript. It allows users to create custom forms by selecting different input types such as Text, Number, Email, Radio, Checkbox, Select, and Submit buttons. Dynamic form builders are commonly used to generate forms without manually coding each field. :contentReference[oaicite:0]{index=0}

## 🎯 How It Works

1. Select a field type from the dropdown.
2. Click the create button.
3. The selected field is added to the form dynamically.
4. Repeat the process to build a custom form.
